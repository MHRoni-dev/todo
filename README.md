## Technology Used
- Language : Typescript
- Framework : NextJs
- Database : MongoDB
- ODM : Mongoose
- Query : Tanstack Query fka React Query, Axios 
- UI : Shadcn 
- Validation : zod

## Getting Started

First, run the development server:

```bash

# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the App.
